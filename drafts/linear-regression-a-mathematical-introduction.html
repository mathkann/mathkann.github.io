def shuffle(df, rand_seed):
    random.seed(rand_seed)
    df['_id'] = [random.random() for i in xrange(len(df))]
    df = df.sort('_id')
    del df['_id']

Evaluation Metric

Root Mean Squared Logarithmic Error (RMSLE) is the metric used in this competition. The RMSLE is calculated as

$\sqrt{\frac{1}{n} \sum_{i=1}^n (\log(p_i + 1) - \log(a_i+1))^2 }$

In [20]:
# Root Mean Squared Logarithmic Error Function
def rmsle(y_pred, y_actual):
    mean_squared_log_error = np.mean(np.power((np.log(y_pred + 1) - np.log(y_actual + 1)), 2))
    rmsle = math.sqrt(mean_squared_log_error)
    return rmsle
